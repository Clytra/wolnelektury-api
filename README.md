# Wolne Lektury API

REST API for Wolne Lektury (target .NET 8).

## Requirements

- [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0) (version pinned in `global.json`)

## Getting started

```powershell
# Restore and build
dotnet restore
dotnet build

# Run 
# dotnet run --project src/WolneLektury.Api
```

## Tests

```powershell
# Run tests 
# dotnet test
```
